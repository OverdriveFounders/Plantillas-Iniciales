# Sabrosito Take Away - Web multipagina SEO local

Proyecto web base para restaurante de comida rapida/take away enfocado en conversion movil y posicionamiento local en Alicante.

## Paginas incluidas
- Home: index.html
- Carta: menu.html
- Pedido online: pedido.html
- Sobre nosotros: sobre-nosotros.html
- Promociones: promociones.html
- FAQ: faq.html
- Contacto: contacto.html
- Legales: carpeta legal/

## SEO local incluido
- H1 y headings orientados a keywords locales.
- NAP visible en bloques clave.
- Menu indexable en HTML.
- Mapa embebido.
- Schema JSON-LD para Restaurant, LocalBusiness, Menu y FAQPage.
- sitemap.xml y robots.txt.

## Estilo y UX
- Mobile-first.
- Header sticky + boton flotante WhatsApp + barra movil de CTA.
- Botones grandes y accesibles.
- Copy corto orientado a conversion.

## Estructura
- assets/css/styles.css
- assets/js/main.js
- docs/propuesta-ux-seo.md

## Como previsualizar
Abre index.html con Live Server o cualquier servidor estatico local.

## Personalizacion rapida
1. Cambia NAP (nombre, direccion, telefono) en todas las paginas.
2. Actualiza ciudad en H1/title/meta/schema.
3. Sustituye imagenes de ejemplo por fotos reales del local y producto.
4. Conecta el formulario de pedido a tu backend o WhatsApp API.
