# Santo Impasto - Web estatica (HTML, CSS, JS)

Proyecto multipagina para pizzeria italiana premium, desarrollado solo con HTML, CSS y JavaScript vanilla.

## Estructura

- index.html
- carta.html
- nosotros.html
- locales.html
- reservas.html
- pedidos.html
- promociones.html
- club.html
- contacto.html
- faq.html
- legal-aviso.html
- legal-privacidad.html
- legal-cookies.html
- 404.html
- assets/css/styles.css
- assets/js/main.js
- assets/js/menu-data.js
- assets/js/carta.js
- sitemap.xml
- robots.txt

## Componentes implementados

- Header fijo con navegacion activa.
- Menu movil con drawer y overlay.
- Hero visual con triple CTA.
- Tarjetas de platos, locales y promociones.
- Slider de testimonios.
- Bloque de fidelizacion.
- Sticky CTA movil.
- Modal de reserva global.
- Banner de cookies configurable.
- Footer completo con enlaces legales.

## Features UX y conversion

- CTA principal de reservas y CTA secundaria de pedidos en todo el sitio.
- Formularios con estado de envio (loading/feedback).
- Carta dinamica por categorias desde JSON en JS.
- FAQ desplegable para reducir friccion.

## SEO local aplicado

- Meta title y meta description por pagina.
- H1 unico por pagina.
- Schema.org de tipo Restaurant en Home.
- Schema.org FAQPage en FAQ.
- sitemap.xml y robots.txt.
- Keywords locales orientadas a: pizza napolitana, pizzeria italiana, delivery, take away, reservas.

## Como usar

1. Abrir index.html en navegador.
2. Para despliegue, subir toda la carpeta a hosting estatico.
3. Actualizar URLs de sitemap, schema, maps y WhatsApp con datos reales de negocio.

## Personalizacion rapida

- Colores y tipografia: assets/css/styles.css
- Navegacion, modal, cookies y slider: assets/js/main.js
- Contenido de carta: assets/js/menu-data.js
- Render de carta: assets/js/carta.js
